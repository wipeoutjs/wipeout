Custom $context.find filters

Find filters (such as $instanceof, $ancestry and $type) are static functions defined on the wo.findFilters object. You can add your own filter by adding a custom method. Custom filters must begin with $ to differenciate them from property filters.

// find a viewmodel based on it's model wo.findFilters.$hasModel = function(currentItem, searchTerm, index) { // searchTerm will be the namespace passed in if (searchTerm) return currentItem.model != null; else return currentItem.model == null; };

And then using the filter

<wo.content-control model--s="Hello wipeout!"> <set-template> <wo.content-control> <set-template> <span wo-content="$context.find({$hasModel: true}).model"></span> </set-template> </wo.content-control> </set-template> </wo.content-control> <wo.view template-id="'custom-find-filters-1'"></wo.view>
