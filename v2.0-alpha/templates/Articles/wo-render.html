The Render attribute

wo-render="value"

data-wo-render="value"

The render attribute renders a piece of data or a view model.

The render attribute is similar to the attibute but is more powerful. As a rule of thumb, you should use content to display simple things such as strings, numbers and simple objects and render to display view models or items which are not surrounded by a html tag {{$this.someValue}}.

<wo.content-control model--s="Hello Wipeout!"> <set-template> <span wo-render="$model"></span> </set-template> </wo.content-control> <wo.view template-id="'wo-render-attr-1'"></wo.view>

Rendering without a html tag

You can also render items without surrounding them by a tag, using the {{value}} notation.

<wo.content-control model--s="Hello Wipeout!"> <set-template> {{$model}} </set-template> </wo.content-control> <wo.view template-id="'wo-render-attr-2'"></wo.view>
