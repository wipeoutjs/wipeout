Set proerties using HTML elements

To set more complex properties or long strings (such as templates) you can set properties as HTML elements rather than HTML attributes

<wo.content-control> <!-- Setting the value within a html element --> <my-object-1>{val: "Hello Wipeout!"}</my-object-1> <!-- Setting the value within a html element attribute --> <my-object-2 value--string="Hello Wipeout!"></my-object-2> <!-- Setting a complex value within a html element --> <my-object-3> <!-- Create an Object. The "js-" part lets wipeout know that the "o" in "object" should be upper case--> <js-object> <!-- Setting the value within a html element with a parser --> <val parser="string"> Hello Wipeout! </val> </js-object> </my-object-3> <set-template> Binding to element value 1: <span wo-content="$this.myObject1.val"></span> <br /> Binding to element value 2: <span wo-content="$this.myObject2"></span> <br /> Binding to element value 3: <span wo-content="$this.myObject3.val"></span> <br /> </set-template> </wo.content-control> <wo.view template-id="'bind-with-elements-script-1'" />
