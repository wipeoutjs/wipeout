Dispose of the binding

There are often subscriptions and other disposable scenarios which occur outside of the property argument to the binding. These must be disposed of manually to prevent memory leaks. In order to dispose of an item manually a dispose function must be returned from the binding function.


Declaration <wo.view model--mybinding="$this.model"></wo.view> Binding: wo.bindings.mybinding = function mybinding (viewModel, property, renderContext) { // observe the model property of the viewModel. This will create a disposable // which we want to dispose of when the binding is torn down var disposable = viewModel.observe(property.name, function (...) { ... }); return function () { disposable.dispose(); }; -- OR -- return disposable; };
