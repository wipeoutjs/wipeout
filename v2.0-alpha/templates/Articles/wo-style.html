The Style attribute

wo-style-styleName="value"

data-wo-style-styleName="value"

The style attribute defines an style on a html element. The style name will be the end of the wo-style- definition.

<wo.content-control model--s="#98FB98"> <set-template> <span wo-style-background-color="$model">Hello wipeout!</span> </set-template> </wo.content-control> <wo.view template-id="'wo-attr-style'"></wo.view>
