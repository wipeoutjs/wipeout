Custom html attributes

You can also add custom html attributes to your application by using the wo.addHtmlAttribute method.

// a custom html attribute which clears the value of an input when it is clicked wo.addHtmlAttribute("clear-when-clicked", function (element, attribute, renderContext) { attribute.onElementEvent("click", renderContext, function () { element.value = ""; }); }); <wo.content-control> <set-template> <input value="click me" wo-clear-when-clicked /> </set-template> </wo.content-control> <wo.view template-id="'wo-custom-html-attribute-1'"></wo.view>

Attribute arguments

Name Description
element The html element which the attributes is set on.
attribute The attribute is a wipeout htmlPropertyValue. It contains functionality for getting setting and watching properties. Full documentation is in the API under
renderContext renderContext is the current render context with $this, $parent etc...
Returns The return value is either a function or an object with a dispose function which will undo anything done in the html attribute. This is important to prevent memory leaks. Any actions using the attribute argument do not need to be disposed of.
