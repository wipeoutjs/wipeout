The Mouseout attribute

wo-mouseout="eventHandler"

data-wo-mouseout="eventHandler"

The mouseout attribute defines an event handler to be fired when a mouse leaves the element.

<wo.content-control model--s="Hello Wipeout!"> <set-template> <input wo-mouseout="logic: alert($this.model)" /> </set-template> </wo.content-control> <wo.view template-id="'wo-mouseout-attr'"></wo.view>
