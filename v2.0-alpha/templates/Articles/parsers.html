Parsers

Within a template, most property setters are not parsed, but rather they are interpreted as javascript code, compiled and bound. This behavior can be altered with property parsers. An example of a property with a default property parser (TODOM link) is id, where:

<wo.view id="myId"></wo.view>

will set the id of the current model to "myId", and not to the value of any myId property.
