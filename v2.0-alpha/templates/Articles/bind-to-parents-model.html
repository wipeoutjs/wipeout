Set the model from the parent's model

In order to keep clean contextual boundaries, the model property of a child viewModel can be bound to a property of its parent's model using the keywords $model.

<wo.content-control model="{ value: 'Hello wipeout!' }"> <set-template> <wo.content-control model="$model.value"> <set-template> Binding to a property on parent's model: <span wo-content="$model"></span> </set-template> </wo.content-control> </set-template> </wo.content-control> <wo.view template-id="'bind-to-parents-model-script-1'"></wo.view>

In this example there is a parent viewModel which has a child viewModel. The model of the child is bound to a property of the model of the parent. $this points to the scope of the viewModel 1 level up
