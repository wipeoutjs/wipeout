Binding to the current model

Unless you are building generic components, most bindings will point to the model. Binding to the model can be done with the $model keyword.

<wo.content-control model="{ value: 'Hello wipeout!' }"> <set-template> Binding to current scope: <span wo-content="$model.value"></span> </set-template> </wo.content-control> Renders: <wo.view template-id="'bind-to-model-script-1'"></wo.view>
$model is shorthand for $this.model
