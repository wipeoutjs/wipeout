Changing the default binding for a property

The default binding for a property can be changed when defining a viewModel.

Say for instance, you wish the model of your viewModel to always bind two ways, you can use the following code:

wo.viewModel("myViewModel") .binding("model", "tw") .build();

Now setting the model of my-view-model inside a template will always bind two ways.
