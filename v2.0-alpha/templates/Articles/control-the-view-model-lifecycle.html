Control the viewModel lifecycle

Viewmodels can be controlled by overriding methods at various stages in the view's lifecycle. You can override a method by simply replacing it on the prototype of the derived class. For example:

var myViewModel = wo.view.extend({ constructor: function myViewModel() { this._super(); }, // override the onRendered function onRendered: function(oldValues, newValues) { this._super(oldValues, newValues); // custom logic here } });

There are quite a few methods to override in order to provide hooks into the viewmodel lifecycle
