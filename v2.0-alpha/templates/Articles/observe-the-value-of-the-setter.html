Observe the value of the setter

If you need to observe the setter for change, you can use the watch function.

Declaration: <wo.view model--mybinding="$this.model"></wo.view> Binding: wo.bindings.mybinding = function mybinding (viewModel, property, renderContext) { // for a regular binding, will return "$this.model" property.watch(function (oldVal, newVal) { // code for when the model value of the parent changes here }, true); // Do any binding stuff here };

The second argument to watch tells wipeout to execute the callback immediately, as if the property had just changed.
