Set proerties using XML elements

As well as setting properties using vm attributes, you can also set them using xml elements. A great example of this is using the template property of a which has been set in most previous examples

<wo.content-control> <set-template> Hello wipeout! </set-template> </wo.content-control> <wo.view template-id="'set-properties-using-xml-elements-script-1'" />
