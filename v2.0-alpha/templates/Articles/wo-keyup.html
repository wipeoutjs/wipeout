The Keyup attribute

wo-keyup="eventHandler"

data-wo-keyup="eventHandler"

The keyup attribute defines an event handler to be fired when a key is released.

<wo.content-control model--s="Hello Wipeout!"> <set-template> <input wo-keyup="logic: alert($model)" /> </set-template> </wo.content-control> <wo.view template-id="'wo-keyup-attr'"></wo.view>
