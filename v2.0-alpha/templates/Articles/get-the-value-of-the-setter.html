Get the value of the setter

To get the setter value use the getter function to create a getter and use the getter to get the value.

Declaration: <wo.view model--mybinding="$this.model"></wo.view> Binding: wo.bindings.mybinding = function mybinding (viewModel, property, renderContext) { // returns a function which can be used to get the value var getter = property.getter(); // returns the value var theValue = getter(); // Do any binding stuff here };

If you are using some asynchronus code (i.e. defining any functions within the binding), it is important to cache the getter. The property.getter() function will only work synchronusly, however the result of the getter function can be invoked at any time

wo.bindings.mybinding = function mybinding (viewModel, property, renderContext) { // cache the getter here var getter = property.getter(); // create an asynchronus function which will be called when the value changes property.watch(function (oldValue, newValue) { // use the getter here var result = getter(); }); };
