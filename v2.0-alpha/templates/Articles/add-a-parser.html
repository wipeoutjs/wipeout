Add a property parser

When creating viewModels, it is sometimes useful to alter how a property is parsed from a template. For instance, you may have a property my-property="$model.value" within a template. This property will be bound to the value property of the model, however you may wish it to be set to the string: "$model.value". For this you can use a default parser.

wo.viewModel("myNamespace.myViewModel") .parser("myProperty", "string") .build();
